# Paradigm IT Services Website — Project Constitution

## 1. Purpose & Vision
Rebuild Paradigm IT Services' web presence to effectively market their IT services to Caribbean SME owners, tech professionals, and diaspora. The site should convey technical competence, Caribbean roots, and forward-thinking innovation ("Engineering the Next Paradigm").

## 2. Project Principles

### Brand Identity
- **Primary Color**: #0D9488 (cyan/teal) — innovation, trust
- **Secondary Color**: #1E3A5F (navy) — professionalism, reliability
- **Typography**: Google Fonts (Roboto)
- **Tagline**: "Engineering the Next Paradigm"
- **Logo Variants**: SVG variants 1, 6, 7 available

### Target Audience
1. Caribbean SME owners (primary)
2. Tech professionals
3. Diaspora (Jamaicans abroad seeking local/regional tech partners)

### Core Value Propositions
- Caribbean/Jamaican tech expertise
- SME-focused solutions
- Innovation and forward-thinking approach

## 3. Technical Boundaries

### In Scope
- Marketing website (serviceparadigm.com or paradigm.com.jm)
- Service showcase
- Contact/inquiry functionality
- Mobile-responsive design

### Out of Scope (Initial Phase)
- E-commerce / payment processing
- Customer portal / dashboard
- Complex integrations (unless specified)

## 4. Success Criteria
- Clear articulation of services and value proposition
- Easy path to contact/consultation
- Professional, modern aesthetic
- Fast load times (performance target)
- Accessible (WCAG 2.1 AA)

## 5. Process Principles
- Elicit requirements before solution design
- Validate with stakeholder before implementation
- Document decisions in spec files
- Phased delivery acceptable
